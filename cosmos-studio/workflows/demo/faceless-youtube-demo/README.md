# Faceless YouTube demo — Plain Signal (tech explainer)

Fully automated demo: every asset (script, narration, visuals, captions, thumbnail, Short) is generated by the pipeline. English, us-en market voice.

## Contents
- `world-bible.md` — preamble + pre-filled episode 1
- `brief.json` — intake brief
- `stills/` — placeholder thumbnail boards

## Run
```bash
node ../../../stages/04_assemble/workflows/faceless-youtube.js brief.json ./output
# outputs: main-video.mp4, short.mp4, thumbnail.webp, description.md, tags.json
```
Runs end-to-end without keys (silent narration + test-pattern visuals, all labeled stubs) — add `ELEVENLABS_API_KEY` + `FAL_KEY` for real output.
